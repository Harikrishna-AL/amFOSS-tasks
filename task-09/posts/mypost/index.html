Making Website using Hugo!!

I would like to share my experiecne making a website using Hugo. Recently, I had to learn to make a website as a task for the recuirtment process for a club called amFOSS. As I was reading about the task, I thought that it would not be possible for me to do it. I only realized that it was a peice of cake only when I started doing it. I decided to do it using Hugo as suggested by the mentors of the club. Making a site using Hugo is quite simple. I only had to install hugo in my system and type a couple of codes to create files and posts. The following code was used to create and initialize a website using Hugo.

C:\Windows\system32>hugo new site <sitename>

after we press enter a folder will be created in the specified location with the name of the site and it will contain the necesssary files which can be edited to modify the site. Now, we need a theme for out website for it to look appealing and cool! Theredore, change the directory to the folder called themes and download the required theme using git command. We can search for themes in the hugo’s website itself. In my case I am using a theme called “Hugo Bootstrap Theme”.

C:\Windows\system32>cd themes
C:\Windows\system32\blog\begineersBlog\themes>git submodule add https://github.com/razonyang/hugo-theme-bootstrap themes/hugo-theme-bootstrap

Now you have to open the congif.toml file and add a line as follows

baseURL = 'https://begineersBlog.github.io/'
languageCode = 'en-us'
title = 'My Blogs'
theme ="hugo-theme-bootstrap"

Now to check the progress type hugo serve in the cmd to locally host it.

C:\Windows\system32\blog\begineersBlog>hugo serve

Then I used the local link to view my website.

To add posts

Adding posts in pretty easy in hugo. Just create a folder called posts in the content folder and inside posts create a .md file of your preferred name. Inside the .md file you can write the content of your post as you preffer.

Adding user profile, social media and menu

All this can be added to website by editing the config.toml file as shown below.

baseURL = 'https://begineersBlog.github.io/'
languageCode = 'en-us'
title = 'My Blogs'
theme ="hugo-theme-bootstrap"

[menu]
    [[menu.main]]  
        identifier = "hari"
        name = "hari"
        url = "https://google.com"
        weight = 10

[author]
    name = "Harikrishna Satheesh Pillai"
    company = "Amrita Vishwa Peetham"
    avatar = "images/logo.webp"
    bio = "Studying Computer Science Engineering at Amrita University \nAiming for amFOSS."
    location = "Mumbai, India"

[social]
    Instagram = "hari393203.com"

Now, that the site is completed we can build the site to publish it using github. For that type the code in cmd as follows.

C:\Windows\system32\blog\begineersBlog>hugo

After you do this, you may notice that a new folder is created in the webdite’s folder named publish. This folder contains all your website files and you can directly upload it to github.
